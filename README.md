# WiFiChallenge Writeups

Repositorio de writeups, apuntes y documentación técnica sobre los laboratorios de WiFiChallenge.

El objetivo de este repositorio es documentar el proceso de análisis y resolución de retos relacionados con seguridad WiFi y protocolos inalámbricos en un entorno controlado y educativo.
